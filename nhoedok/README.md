# 📚 N회독 플래너

프로젝트 → 과목 → 항목 3단계로 학습을 관리하고, 달력에 드래그로 계획을 세우는 앱입니다.

## 주요 기능

- **프로젝트 관리** — 상단 탭에서 여러 프로젝트 전환 (더블클릭으로 편집)
- **과목 & 항목** — 프로젝트마다 과목, 과목마다 항목 추가 (더블클릭으로 이름 편집)
- **회독 카운트** — ✓ 버튼으로 회독 수 증가, − 로 감소
- **계획 달력** — 항목을 꾹 눌러 달력으로 드래그하면 학습 계획 등록
- **자동 저장** — localStorage에 자동으로 저장됨

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 기술 스택

- React 18
- Vite 5
- 순수 inline CSS (외부 CSS 라이브러리 없음)
